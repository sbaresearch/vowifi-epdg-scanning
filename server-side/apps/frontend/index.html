VoWiFi Watchdog

Project VoWiFi Watchdog aims to monitor the global VoWiFi landscape with automated scannning and analysis of worldwide ePDG infrastructure and IKEv2 DH handshakes. It's main goal is to provide insights accessable to all, researchers, operators and the general public in an intuitive and interactive way.

Overview

What this is all about

This project is based on the paper Diffie-Hellman Picture Show: Key Exchange Stories from Commercial VoWiFi Deployments1. This project runs an automated pipeline that discovers ePDG infrastructure, then executes multiple IKEv2 handshake tests and exposes the Data through an API and map UI.

  1. Gabriel K. Gegenhuber, Florian Holzbauer, Philipp É. Frenzel, Edgar Weippl, and Adrian Dabrowski. 2024. Diffie-hellman picture show: key exchange stories from commercial VoWiFi deployments. In Proceedings of the 33rd USENIX Conference on Security Symposium (SEC '24). USENIX Association, USA, Article 26, 451–468.↩

Discovery

Generates all possible ePDG domains with MCC and MNC combinations and then runs DNS scans to identify active VoWiFi targets.

Scanning

Multiple IKEv2 handshake variants are made against discovered targets to evaluate operator support, downgrade tolerance, and response quality.

Storage

Scan results are normalized, analyzed and stored for later serving via API and visualization.

Visualization

Visualizations are provided to give intuitive insights into the global VoWiFi landscape, with a special focus on handshake support.

Map boundaries are based on Natural Earth data. This project also relies on zdns and MCC/MNC enrichment sources.
